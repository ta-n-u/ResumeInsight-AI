Core Features

Everything HR teams need

Built with NLP and machine learning to make every step of resume screening faster, fairer, and more accurate.

NLP Resume Parsing
Automatically extracts Skills, Education, and Experience from PDF resumes using spaCy Named Entity Recognition.
Semantic Matching
Uses word embeddings and Cosine Similarity to match resumes to job descriptions — beyond simple keyword counting.
Skill Gap Analysis
Visual Radar/Spider charts showing exactly which required skills a candidate has and which are missing.
Blind Screening
Toggle to hide names, emails and phone numbers — replacing them with anonymous IDs for bias-free evaluation.
Candidate Ranking
Automatically ranks all candidates by their match percentage so the best fits rise to the top immediately.
Secure & Role-Based
HR Manager and Super Admin roles with secure authentication and PII protection for all candidate data.

About This Project

Built for Final Year CSIT

ResumeInsight AI is an intelligent resume screening and skill gap analysis system developed as a Final Year Project for the Bachelor of Science in Computer Science and Information Technology (B.Sc. CSIT) program.

The system addresses the challenge HR departments face when dealing with high volumes of resumes — slow manual screening, inconsistent evaluation, and unconscious bias. ResumeInsight AI automates the process using NLP and machine learning to make hiring smarter and fairer.

The matching engine goes beyond simple keyword counting by using word embeddings and Cosine Similarity to understand contextual meaning — recognizing that "Developer" and "Engineer" are semantically similar roles.
